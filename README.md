Project Overview
This project was completed as part of the Digital Literacy course. The aim of this project is to develop essential digital skills required for academic and professional environments. It focuses on understanding digital tools, maintaining a professional online presence, practicing safe internet usage, and improving communication skills.

The project consists of five tasks, each covering a different aspect of digital literacy. All tasks are organized in this repository along with supporting files, screenshots, and a detailed project report.
